## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Visit [http://localhost:3000](http://localhost:3000) to see the landing page.

## Design Principles

The landing page follows Diagrammatic's design principles:

1. **Clean & Modern:** Minimalist design with focus on content
2. **Gradient Accents:** Subtle gradients for visual interest
3. **Dark Mode Support:** Full dark mode implementation
4. **Responsive:** Mobile-first, responsive design
5. **Accessible:** WCAG compliant with semantic HTML
